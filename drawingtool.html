Art Tool is a digital painting tool made in Phaser 3 using Javascript. The tool allows for rapid creation of abstract patterned images. Users can explore a mixture of standard drawing tools (brush and canvas color, brush size, erasers) along with various automated and distorted effects.

Effects include: automated scaling of brush size, rotation of brush location, scaling and transparency control of overlayed canvases, and painting with the brush texture set to the current canvas image.

All visual assets were created in Gimp, and audio created in Logic Pro.

Art Tool is avaliable to use on Itch.io


I created 24 different buttons that interact in various ways, setting the logic for each button to account for various other effects that might already be turned on. When the mouse is clicked and dragged over the canvas element of the interface, the texture/opcacity of the brush stroke is determined by the state of the buttons. The tool then accounts for the states of automated effect buttons (which can automated scaling, rotation, mirroring, movement) as it draws the brush stroke to the screen. Over 90 UI assets were created by hand to complete the control panel and give buttons various appearances while they are in different states.
